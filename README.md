## Keyboard Shortcuts

| Shortcut | Action |
| --- | --- |
| <kbd>1</kbd> | Toggle symbols |
| <kbd>2</kbd> | Toggle digits |
| <kbd>3</kbd> | Toggle lowercase letters |
| <kbd>4</kbd> | Toggle uppercase letters |
| <kbd>A</kbd> | Select all/none |
| <kbd>F</kbd> | Fullscreen mode |
| <kbd>D</kbd> | Dark mode |
